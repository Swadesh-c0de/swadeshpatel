# Swadesh Patel

A minimalist, high-performance developer portfolio built with Next.js 16 and Tailwind v4.

---

## Technical Overview

This portfolio is engineered for maximum performance and an understated, premium user experience. It leverages modern web standards and cutting-edge frameworks to deliver a seamless interface.

### Key Features

*   **Custom Page Transitions**: Flicker-free cross-fade navigation orchestrated with Framer Motion.
*   **Modern Design System**: A cohesive glassmorphism aesthetic featuring bento-grid layouts and consistent micro-interactions.
*   **Performance Optimization**: 
    *   **Turbopack Integration**: Leveraging Next.js 16 for near-instant builds and HMR.
    *   **Lazy Loading Strategy**: Optimized for heavy assets like the dChat interactive demo.
    *   **Tree-shaken Iconography**: Intelligent imports from Lucide and React Icons to minimize bundle size.

### Tech Stack

*   **Framework**: Next.js 16 (App Router)
*   **Library**: React 19
*   **Styling**: Tailwind CSS v4
*   **Animations**: Framer Motion
*   **Icons**: Lucide React & React Icons
*   **Utility**: clsx & tailwind-merge

---

## Project Highlights

*   **Vibe-Fi**: A terminal-based music player (TUI) featuring a retro-futuristic audio visualizer and YouTube integration.
*   **dChat**: A real-time communication platform showcasing seamless iframe integration and state management.
*   **Research & Archive**: A collection of dedicated technical explorations, including Arch Linux ricing and system-level configurations.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start development server**:
    ```bash
    npm run dev
    ```

3.  **View locally**: 
    Open [http://localhost:3000](http://localhost:3000)
